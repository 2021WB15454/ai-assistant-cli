# AI Productivity CLI Assistant

This assistant helps you manage tasks, schedule events, send emails, get weather/news updates, and summarize content using OpenAI APIs.

## Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Fill in API keys in `config.py`.

3. Run:

```bash
python main.py
```

## Commands
- task
- calendar
- email
- weather
- news
- summarize
- chat
- exit
